# Easy Emulation

> With only git and docker installed, bring up an emulator on any platform.

```shell
git clone https://github.com/Opentrons/opentrons-emulation
cd opentrons-emulation/compose
docker-compose up
```

> Bring down `ctrl c`

> Make sure down `docker-compose down`
